# MU Parent Portal - Zen Edition

A premium, card-driven mobile experience for parents to track their ward's academic journey at MU. This application moves away from traditional tab-bars to a "Zen" navigation system focused on clarity, visual insights, and proactive communication.

## ✨ Key Features

### 🗂️ Card-Driven Navigation (The Zen Dashboard)
The application replaces traditional bottom navigation with an interactive card grid. 
- **Intelligent Status Indicators**: Cards for Schedule, Assignments, and Grades display real-time status (e.g., "Action Needed," "Danger," "All Good") based on student data.
- **One-Tap Access**: Navigate directly to detailed views with a single tap.

### 📊 Visual Progress Trends
Experience data, don't just read it.
- **Interactive SVG Charts**: Toggle between GPA Trends and Attendance Patterns using an animated SVG line graph.
- **Dynamic Gradients**: Visual themes shift based on the metric being viewed (Purple for GPA, Green for Attendance).

### 🔔 Unified Alert System
Never miss a critical update.
- **Contextual Synchronization**: Alerts seen on the Dashboard (like overdue assignments or new grades) follow you into the relevant detail pages.
- **Priority Coding**: Color-coded alerts for Danger (Red), Warning (Orange), and Info (Purple).

### 🗞️ Campus Integration
Stay connected to the University ecosystem.
- **Life @ MU**: A vibrant feed of campus events and student highlights.
- **Official Announcements**: Dedicated space for administration notices with priority badges.

## 🎨 Design Philosophy
- **Zen Navigation**: A minimal, distraction-free environment.
- **Glassmorphism**: Modern, frosted-glass components for a premium feel.
- **Micro-Animations**: Subtle transitions and SVG drawing effects ensure the app feels alive and responsive.
- **Mobile-First**: High-touch targets and scroll-aware layouts optimized for handheld use.

## 🛠️ Technology Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Modern CSS Variables, Flexbox, Grid)
- **Visuals**: Hand-coded SVG Paths and Linear Gradients

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AkashSharma19/CoachParentMobileAppWireframe.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
